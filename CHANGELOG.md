# Changelog

## Unreleased

## 0.1.0 — 2026-07-16

- Подготовлен первый публичный выпуск Thermex Key Exporter.
- Добавлен QR-only экспорт `device_id` и `local_key` из Thermex Home.
- Готовые сборки не требуют APK, Python, ADB или подключения телефона к
  компьютеру.
- Поддерживаются Windows x64, macOS ARM64 и Linux x64.
- Нативные Release-архивы содержат macOS GUI, Windows GUI и CLI, а также
  Linux CLI. GUI-bundle не перенаправляет аргументы в CLI.
- В сборки включён собственный набор корневых сертификатов для надёжного
  защищённого соединения с облаком Thermex.
- CLI печатает QR-код непосредственно в терминале и не создаёт временный PNG.
- Pillow используется только GUI; базовой CLI-установке он не требуется.
- GitHub Actions автоматически готовит черновик Release с проверенными
  платформенными архивами и единым файлом контрольных сумм.
